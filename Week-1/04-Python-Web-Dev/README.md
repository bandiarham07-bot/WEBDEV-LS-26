## Why This Topic

Before we jump into building websites with Django, we need to get the core Python basics down. Go through this crash course to get familiar with how the language works. Pay special attention to the sections on Functions, Lists, and Dictionaries—you will be using dictionaries constantly to send data to your web pages later!

---

## What You Will Learn

- **Python Syntax**
- **Lists and Dictionaries**
- **Object oriented Programming** - Django relies heavily on 'Classes' to build databases and handle web traffic. If you don't know what a Class is, Django will look like total magic and be really confusing
